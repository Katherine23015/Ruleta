Ruleta-black cat
Es un prototipo funcional de un sistema de ruleta europea desarrollado en Java. 

Descripción del Proyecto
Este software permite a los usuarios interactuar con una simulación de ruleta, realizar apuestas de probabilidad equilibrada (rojo/negro, par/impar) y realizar un seguimiento detallado de su desempeño a través de un sistema de historial y estadísticas.

Características Principales
Generación aleatoria de números entre el 0 y 36
Sistema para procesar montos y tipos de apuesta dinámica, es decir, apuestas de Par/Impar y Rojo/Negro.
Cálculos automáticos de detallado de rondas jugadas, porcentaje de acierto y balance neto (ganancia/pérdida).
